# 欢迎来到我的博客

这是我的第一篇博客文章。在这里我会分享一些技术心得和生活感悟。

## 关于这个博客系统

这个博客系统使用Vue.js构建，支持Markdown格式的文章。每篇文章都需要在开头包含元数据信息：

- **时间**：文章发布日期
- **网页标题**：文章标题
- **作者**：文章作者

## 主要特性

**技术特性：**
- 响应式设计
- 暗色主题支持
- Markdown渲染
- 简洁的界面

**内容特性：**
- 支持多种Markdown语法
- 自动生成文章预览
- 按时间排序显示

## 使用方法

1. 在`public/blog/`目录下创建`.md`文件
2. 在文件开头添加必需的元数据
3. 使用标准Markdown语法编写内容
4. 系统会自动解析并显示文章

希望大家喜欢这个简单的博客系统！
